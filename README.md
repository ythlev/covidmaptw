
# covidmaptw

<!-- badges: start -->
<!-- badges: end -->

This package is used to create an svg map of COVID-19 infection rates in Taiwan.

## Installation

You can install covidmaptw from GitHub with:

``` r
remotes::install_github("ythlev/covidmaptw")
```

## Example

``` r
library(covidmaptw)
make_covid_map(29MAY.svg)
```
